1

إنشاء مستودع على GitHub

  • اذهب إلى github.com
  • اضغط على "New Repository"
  • اسم المستودع: portfolio
  • ⚠️ مهم: لا تضع علامة على "Add README"
  • اضغط "Create repository"
🔗 رابط GitHub: https://github.com
2

ربط المشروع بـ GitHub

استخدم هذه الأوامر في Terminal:

git remote add origin https://github.com/hanyarafaosman/portfolio.git
git branch -M main
git push -u origin main
💡 تأكد من تشغيل الأوامر في مجلد المشروع
3

تفعيل GitHub Pages

  • اذهب إلى مستودعك على GitHub
  • اضغط على تبويب "Settings"
  • انزل إلى قسم "Pages"
  • اختر "Deploy from a branch"
  • اختر "main" branch و "/ (root)"
  • اضغط "Save"
